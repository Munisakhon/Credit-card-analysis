# 💳 Credit Card Customer Analytics Dashboard

An interactive **Streamlit** web application for analyzing credit card customer behavior, visualizing customer attrition (churn), and evaluating customer risk using the **BankChurners Dataset**.

## 🌐 Live Demo

**Application:** https://credit-card-analysis-iokedfvtnu3rvbcmmmasny.streamlit.app/

---

# 📖 Project Overview

This project was developed to analyze credit card customer data and provide an interactive dashboard for business intelligence.

The application enables users to:

* 📊 Analyze customer demographics
* 💳 Explore credit card information
* 📈 Visualize transaction behavior
* 🔍 Filter and search customers
* ⚠️ Estimate customer churn risk
* 📋 View statistical summaries of the dataset

The project is completely built using **Python** and **Streamlit**.

## 🔮 Customer Risk Assessment

A simple rule-based scoring system estimates customer churn risk based on:

* Transaction count
* Transaction amount
* Inactive months
* Customer service contacts
* Credit limit
* Relationship count

Risk levels:

* 🟢 Low Risk
* 🟡 Medium Risk
* 🔴 High Risk

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/credit-card-analysis.git
```

Go to the project folder

```bash
cd credit-card-analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
